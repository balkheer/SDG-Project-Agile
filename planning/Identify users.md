1- User

The main user of the application.
The user can register, log in, view posts, like and comment on posts, receive notifications, and send emergency alerts.


2- Admin

The administrator manages the system.
The admin can manage users, approve or delete posts, and monitor system activities.

3- Seller

The seller can post apartment listings and upload images.
They can manage their posts and interact with users through comments.

4- System

The system handles internal operations automatically.
It sends notifications, processes user actions, and manages data within the application.
The system includes an AI chatbot that helps users search for apartments and provides recommendations.
It responds to user queries and assists in finding suitable listings.
